# ✅ 简待办 (Simple Todo)

一个基于 Android Jetpack Compose 开发的现代化待办事项应用。极简设计，功能强大。

## ✨ 功能特点

*   🎨 **多彩标签**：支持 6 种颜色标记任务优先级。
*   📅 **时间管理**：精确到分钟的开始与结束时间设置。
*   🌗 **深色模式**：完美适配 Android 系统深色/浅色主题切换。
*   💾 **自动保存**：退出应用数据不丢失（本地持久化存储）。
*   ✏️ **灵活编辑**：随时修改内容、时间和颜色。
